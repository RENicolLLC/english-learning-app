# English Learning App

A comprehensive React-based English learning platform with AI-powered features, scenario-based learning, and pronunciation practice.

## 🚀 Features

### 1. Learning Content
- 5000+ unique daily scenarios
- Vocabulary building exercises
- Grammar practice modules
- Medical & professional scenarios
- Cultural context integration

### 2. Interactive Learning
- Real-time pronunciation feedback
- AI-powered conversations
- Speech recognition
- Progress tracking
- Personalized learning paths

### 3. User Experience
- Responsive design (mobile & desktop)
- Multi-language support
- Offline capability
- Dark/Light mode
- Accessibility features

### 4. Technical Features
- Supabase authentication
- Firebase integration
- Error handling & monitoring
- Analytics tracking
- Performance optimization

## 🚀 Quick Start

1. **Clone & Install**
   ```bash
   git clone [your-repo-url]
   cd english-learning-app
   npm install
   ```

2. **Environment Setup**
   ```bash
   cp .env.example .env
   ```
   Fill in your API keys:
   - Supabase credentials
   - Firebase config
   - OpenAI API key
   - Google Cloud API key

3. **Start Development**
   ```bash
   npm start
   ```
